# manage_system

数据库课设。

## 项目结构

```
manage_system
|-- app : 服务路由等等
|-- config : 项目配置
|-- dbconn : 数据库连接
|-- resource : 运行时资源
`-- sql : 数据库构建代码
```

## 提供功能

1. 基本
	1. 登录、登出
	2. 修改密码
2. 用户管理
	1. 用户添加
	2. 密码重置
	3. 用户删除
	4. 修改权限
3. 品类管理
	1. 列出品类
	2. 添加品类

4. 商品管理
	1. 列出批次
	2. 添加批次
